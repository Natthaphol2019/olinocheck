# 🎯 ระบบกะงานแบบง่าย (Simplified Shift System)

## 📌 สรุปการปรับปรุง

ปรับระบบกะงานให้เป็นแบบ **ง่ายและตรงไปตรงมา** - พนักงานเลือกเองได้เลย ไม่ต้องให้ HR จัดให้

---

## ✨ การเปลี่ยนแปลง

### ❌ **สิ่งที่ลบออก:**
1. ❌ ระบบ `employee_shifts` (HR จัดกะให้พนักงาน)
2. ❌ Auto-select กะจาก employee_shifts
3. ❌ หน้า "กะงานของฉัน" (MyShiftSchedule)
4. ❌ ฟีเจอร์จัดกะงานให้พนักงานทีละคน/หลายคน
5. ❌ Trigger จัดกะอัตโนมัติให้พนักงานใหม่

### ✅ **สิ่งที่เก็บไว้:**
1. ✅ ตาราง `shifts` (ข้อมูลกะงาน)
2. ✅ Trigger ตรวจจับสายอัตโนมัติ
3. ✅ การเลือกกะตอนเช็คอิน (UI แบบ Card)
4. ✅ Late detection ตาม grace period
5. ✅ OT calculation

---

## 🕐 กะงานเริ่มต้น 2 กะ

### **กะเช้า (Morning Shift)**
```
เวลาเข้า:     09:30 น.
เวลาเลิก:     19:30 น.
เวลา OT:      19:30 - 23:30 น.
สายหลังจาก:   09:45 น. (หลังจาก 15 นาที)
สี:           น้ำเงิน (#3B82F6)
```

### **กะเย็น (Evening Shift)**
```
เวลาเข้า:     11:30 น.
เวลาเลิก:     21:30 น.
เวลา OT:      21:30 - 23:30 น.
สายหลังจาก:   11:45 น. (หลังจาก 15 นาที)
สี:           ม่วง (#8B5CF6)
```

---

## 🎨 วิธีใช้งาน

### **สำหรับพนักงาน:**

#### **1. เช็คอิน/เอาท์:**
```
1. เปิดหน้า "เช็คอิน/เอาท์"
2. เลือกกะงาน (Card สวยงาม):
   ☐ กะเช้า (09:30-19:30)
   ☐ กะเย็น (11:30-21:30)
3. กรอกเวลาเข้า-ออก
4. บันทึก
```

#### **2. เลือกกะงาน:**
```
- คลิกที่ Card กะที่ต้องการ
- Card จะเปลี่ยนสี (gradient ตามสีของกะ)
- แสดง ✓ ว่าเลือกแล้ว
- ข้อมูลครบ: เวลา, OT, สายหลังจาก
```

---

### **สำหรับ HR/Admin:**

#### **1. ดูข้อมูลกะงาน:**
```
เมนู → จัดการกะงาน (/manager/shifts)

แสดง:
✅ กะเช้า - 09:30-19:30 (OT 19:30-23:30)
✅ กะเย็น - 11:30-21:30 (OT 21:30-23:30)
✅ เวลาเข้า-ออก, OT, grace period
✅ สถานะเปิดใช้งาน
```

#### **2. เพิ่ม/แก้ไขกะงาน:**
```
ทำใน Supabase Dashboard → shifts table
หรือรัน SQL: simple_shifts_seed.sql
```

## 📋 ขั้นตอนการติดตั้ง

### **1. รัน SQL Seed Data:**

เปิด Supabase SQL Editor แล้วรัน:

```sql
-- ไฟล์: simple_shifts_seed.sql

-- กะเช้า: 09:30-19:30
INSERT INTO shifts (
    name, start_time, end_time, 
    ot_start_time, ot_end_time, 
    grace_period_minutes, color, 
    display_order, description, is_active
)
VALUES (
    'กะเช้า',
    '09:30:00',
    '19:30:00',
    '19:30:00',
    '23:30:00',
    15,
    '#3B82F6',
    1,
    'กะเช้า: เข้างาน 09:30 - เลิกงาน 19:30 (OT 19:30-23:30)',
    TRUE
);

-- กะเย็น: 11:30-21:30
INSERT INTO shifts (
    name, start_time, end_time, 
    ot_start_time, ot_end_time, 
    grace_period_minutes, color, 
    display_order, description, is_active
)
VALUES (
    'กะเย็น',
    '11:30:00',
    '21:30:00',
    '21:30:00',
    '23:30:00',
    15,
    '#8B5CF6',
    2,
    'กะเย็น: เข้างาน 11:30 - เลิกงาน 21:30 (OT 21:30-23:30)',
    TRUE
);
```

### **2. ตรวจสอบ:**
```sql
SELECT * FROM shifts ORDER BY display_order;
```

ควรเห็น 2 แถว: กะเช้า และ กะเย็น

```
┌─────────────────────────────────────┐
│      เลือกกะงาน *                    │

```

### **หลังเลือกแล้ว:**

```
┌─────────────────────────────────────┐
│  ✓ กะเช้า (selected)                 │
│  เวลา: 09:30-19:30                   │
│  OT: 19:30-23:30                     │
│  สายหลังจาก: 09:45 น.               │
└─────────────────────────────────────┘

Card เปลี่ยนเป็นสีน้ำเงิน มี ✓
```
